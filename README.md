# STDISCM-KeyboardWarrior

## To play online:

stdiscm-keyboardwarrior.web.app

## To Run Locally:
Step 1: `npm install`

Step 2: `npm run serve`

Step 3: go to localhost:8080

## Add-ons/Dependencies
### Database
Firebase (for Firestore)

### HTTP Client/API
Axios

### Styling
Bootstrap-Vue

Boostrap

SASS

### Routing
vue-router

### States
vuex

### Cookies
vuex-persist
